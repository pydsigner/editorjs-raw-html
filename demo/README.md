# Demo — Editor.js Raw HTML Tool

A tiny demo to try the `RawHtmlTool` from `src/index.js`.

Run locally with one of these commands from the project root:

```bash
# Using the included npm script (requires network to fetch 'serve')
npm run demo

# Or using npx http-server:
npx http-server demo -c-1

# Or using Python's simple server:
python3 -m http.server 8000 --directory demo
```

Then open http://localhost:5000 (for `serve`) or the port shown by the other servers.
